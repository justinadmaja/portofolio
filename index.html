Lake of Shadows (Solo)

Game Description
A 2D pixel art side-scrolling resource management with some action, looting, and a bit of exploration that mainly focuses on a small area. Defend yourself against the terror of the night inside your settlement by employing some villagers and give them equipment to further strengthen your defenses.

The first game project I published. The mobile version (.apk) of this game is currently being optimized and will be released soon on Play Store.

Dungeon of Karma (Solo)

Game Description
Random Generated Dungeon, Top-Down, Pixel-Art, Roguelike, Looting game. In Dungeon of Karma, you build up karma and spend them for permanent upgrades in this roguelike random generated dungeon. Loot, shoot and slash your way through the dungeon to get to the boss room from multiple playthroughs.

With the experience I got from making my first game, I tried something new which is the roguelike randomness of an rpg like game.

FitnessZ (Solo)

Application Description
An application design for a fitness app that help users to manage their time, activities and diet. It comes with some training courses and videos, activity tracker like sleep, calories, steps and water, with calories calculator and food recipes.


A project that I made alone using Figma, a popular online UI design tool which made designing UIs easier.

Danbam Website (Group)

Website Description

I specifically worked on all of the website aspects including the designs, coding and art (logo).
